Welcome to 3D COLREGS

This course is designed to give you a real world experience on understanding and implementing COLREGS.

Let's get started with a brief introduction and guide.

This course consists of levels depicting various scenarios which allow users to interpret COLREGS.

In order to assist the users to have complete situational awareness, there are various tools available for each level, which the user can interact with.

Before you begin:

The Main video consists of a normal view from inside the navigation bridge of own vessel.

Own vessel in all scenarios is a Power Driven vessel with length >50m.

The duration of Main video is 1 minute.

You must completely watch this main video before you are able to proceed to the next stage, which is Assessment.

Toolbar:

There are additional videos available on the bottom toolbar as follows:

Port side view:

This camera gives you a view from the Port side Bridge wing. This allows you to check for any traffic or relevant information on vessel’s Port side, if available.

ECDIS Information:

ECDIS information gives an image displaying the current situation on an ECDIS Chart. Here you will find information related to vessels navigation and ENC’s giving you information such as sea room available, hazards, route information etc. This image is a static image and users cannot interact with this image.

Binocular view:

This video clip will show a close-up of the target, as you would see on a vessel.Using this view, you will be able to clearly see the Day shape/Lights etc of the Target vessel.

Bridge Top View (Monkey Island view):

This view shows you the target as when viewed from Monkey Island. This view allows users to have a less restricted view of the Target than possible from the Main Bridge view.

User Guidelines.

Stern View:

This view shows you the view from the top of the funnel looking towards the stern, while allowing you to get better situational awareness of the scenario. 

Objectives:

Here you can view the Objectives for this particular level of the Course.

Radar Image:

Using this image, you can view the Radar information available for the Target. The user must study this image carefully to apprise himself/herself of the situation and make a mental note of important information such as CPA/TCPA, Range and Bearing, BCR/BCRT etc. This information will form part of the Assessment. Similar to ECDIS, this is a static image, and users cannot interact with this image.

Starboard side view:

This camera gives you a view from the Starboard side Bridge wing. This allows you to check for any traffic or relevant information on vessel’s Starboard side, if available.

Bridge view:

When you click on any of the above icons, you must click on the Bridge Icon to return to the Main video.

Important: When a user clicks on any of the other Icons, the Main Video is paused and will only resume when the user clicks on the Bridge view Icon again. This allows users to watch the scenario completely without losing any information.

Notes:

  • Users are unable to fast forward any videos.
  • Users do not have to watch any of the other informational videos/images completely.
  • Viewers MUST complete the Bridge video in its entirety to proceed to the next Stage.

Assessment:

After you complete watching the Main Level video, you will be asked to complete an assessment for the level before you can proceed to the next level.

Assessment consists of MCQ (Multiple Choice Questions), MCA (Multiple Correct Answers) and True/False.

  • MCQ (Multiple Choice Questions): This type of questions consists of 4 options with only ONE correct answer.
  • MCA (Multiple Correct Answers): This type of questions consists of 4 options with more than one correct answer. Candidates must select all the correct answers to pass.
  • True/False: Self-explanatory. Users must answer if the statement in the question is True or False.

Each Question is allotted a time of 1 minute, failing which it will be considered as a failed attempt

You MUST answer each question correctly before you can proceed to the next question.

Example of Assessment:

  • Level 01 has a total of 10 questions for Assessment.
  • Candidate completed first 6 questions successfully on 1st attempt.
  • On the 7th question he answered incorrectly on first attempt. Candidate is still allowed one more attempt at the same question.
  • If the second attempt is also incorrect, then the candidate will be redirected to the Level video again. 
  • Once he watches the Level video he will be allowed to ‘Resume’ the assessment from the same 7th Question and that attempt will be considered as the 3rd attempt.
  • If the candidate answers incorrectly even on the 3rd attempt, he is still allowed to attempt the same question for the 4th time.
  • However, if the candidate answers incorrectly even on the 4th attempt, he will be redirected to the Main Level video again.
  • After the video, the Assessment is reset and the candidate has to start the assessment all over again, with a different set of 10 questions.
  • No points are awarded for any subsequent attempts, even the Assessment is reset after the 4th incorrect attempt.

Scoring:

  • Each Level is allotted a score of 100 points divided equally by the number of questions for that level.
  • If a candidate answers correctly on his first attempt, he will score the maximum points available for that question.
  • If a candidate scores correctly on second attempt, then he will score only 50% of the points available for that question.
  • If a candidate answers incorrectly on 2nd attempt, he will be redirected to watch the Level video again. Upon completing the video, he will be redirected to ‘resume’ the Assessment from where he had left off.
  • However, he will receive no points on answering questions correctly on 3rd or subsequent attempts.

Example of scoring:

  • Level 01 has a total of 10 questions for Assessment. So, each question is worth 10 points.
  • If he answers correctly on first attempt, he will score 10 points for that question.
  • If he answers correctly on second attempt, he will score 7.5 points for that question.
  • If he answers correctly on third attempt, he will score 6 points for that question.
  • If he answers correctly on fourth attempt, he will score 4.5 points for that question.
  • No points are awarded for 5rd or subsequent attempts.
